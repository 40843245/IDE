# How to change settings of compiler of Kotlin in Eclipse?
## Step
step 1:

In Navigation bar, select 

    Windows -> Preferences

step 2:

Select 

    Kotlin -> Compiler

## settings about Kotlin Compiler
About all settings about Kotlin Compiler, they can set

    1. JVM target version.
    2. JDK Home.
